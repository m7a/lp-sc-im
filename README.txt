`mdvl-sc-im` package for sc-im, an improved spreadsheet calculator based on sc.
See <https://github.com/andmarti1424/sc-im>.
This package builds sc-im with support for libxlsxwriter, see
<https://github.com/jmcnamara/libxlsxwriter>

A colorscheme for use with MDVL systems is included in file `scimrc`.
Install it as follows:

  $ mkdir $HOME/.config/sc-im
  $ cp /usr/share/doc/mdvl-sc-im/scimrc $HOME/.config/sc-im

sc-im uses a 4-clause BSD license. For simplicity, the Ma_Sys.ma-contributed
contents from this repository are placed under CC0, see file `build.xml` for
details.
